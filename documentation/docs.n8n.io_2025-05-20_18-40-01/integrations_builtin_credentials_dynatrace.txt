Dynatrace credentials
#
You can use these credentials to authenticate when using the
HTTP Request node
to make a
Custom API call
.
Prerequisites
#
Create a
Dynatrace
account.
Related resources
#
Refer to
Dynatrace's API documentation
for more information about authenticating with the service.
This is a credential-only node. Refer to
Custom API operations
to learn more. View
example workflows and related content
on n8n's website.
Using Access Token
#
To configure this credential, you'll need:
An
Access Token
Refer to
Access Tokens
on Dynatrace's website for more information.
