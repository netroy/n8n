Groq credentials
#
You can use these credentials to authenticate the following nodes:
Groq Chat Model
Prerequisites
#
Create a
Groq
account.
Supported authentication methods
#
API key
Related resources
#
Refer to
Groq's documentation
for more information about the service.
View n8n's
Advanced AI
documentation.
Using API key
#
To configure this credential, you'll need:
An
API Key
To get your API key:
Go to the
API Keys
page of your Groq console.
Select
Create API Key
.
Enter a
display name
for the key, like
n8n integration
, and select
Submit
.
Copy the key and paste it into your n8n credential.
Refer to
Groq's API Keys documentation
for more information.
Groq API keys
Groq binds API keys to the organization, not the user.
