OpenCTI credentials
#
You can use these credentials to authenticate when using the
HTTP Request node
to make a
Custom API call
.
Prerequisites
#
Create an
OpenCTI
developer account.
Authentication methods
#
API key
Related resources
#
Refer to
OpenCTI's documentation
for more information about the service.
This is a credential-only node. Refer to
Custom API operations
to learn more. View
example workflows and related content
on n8n's website.
Using API key
#
To configure this credential, you'll need:
An
API Key
: To get your API key, go to your
Profile > API access
. Refer to the OpenCTI
Integrations Authentication documentation
for more information.
