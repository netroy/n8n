KoboToolbox credentials
#
You can use these credentials to authenticate the following nodes:
KoboToolbox trigger
KoboToolbox
Prerequisites
#
Create a
KoboToolbox
account.
Supported authentication methods
#
API token
Related resources
#
Refer to
KoboToolbox's API documentation
for more information about the service.
Using API token
#
To configure this credential, you'll need:
An
API Root URL
: Enter the URL of the KoboToolbox server where you created your account. For the Global KoboToolbox Server, use
https://kf.kobotoolbox.org
. For the European Union KoboToolbox Server, use
https://eu.kobotoolbox.org
.
An
API Token
: Displayed in your
Account Settings
. Refer to
Getting your API token
for more information.
