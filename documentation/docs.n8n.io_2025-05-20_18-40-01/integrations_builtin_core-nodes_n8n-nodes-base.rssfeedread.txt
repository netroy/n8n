RSS Read
#
Use the RSS Read node to read data from RSS feeds published on the internet.
Node parameters
#
URL
: Enter the URL for the RSS publication you want to read.
Node options
#
Ignore SSL Issues
: Choose whether n8n should ignore SSL/TLS verification (turned on) or not (turned off).
