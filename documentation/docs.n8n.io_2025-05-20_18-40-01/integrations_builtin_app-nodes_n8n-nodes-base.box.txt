Box node
#
Use the Box node to automate work in Box, and integrate Box with other applications. n8n has built-in support for a wide range of Box features, including creating, copying, deleting, searching, uploading, and downloading files and folders.
On this page, you'll find a list of operations the Box node supports and links to more resources.
Credentials
Refer to
Box credentials
for guidance on setting up authentication.

#
File
Copy a file
Delete a file

Get a file
Search files
Share a file
Upload a file

Create a folder
Get a folder
Delete a folder
Search files
Share a folder

What to do if your operation isn't supported
#
If this node doesn't support the operation you want to do, you can use the
HTTP Request node
to call the service's API.
You can use the credential you created for this service in the HTTP Request node:
In the HTTP Request node, select
Authentication
>
Predefined Credential Type
.
Select the service you want to connect to.
Select your credential.
Refer to
Custom API operations
for more information.
