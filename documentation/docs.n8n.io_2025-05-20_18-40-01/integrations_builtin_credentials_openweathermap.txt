OpenWeatherMap credentials
#
You can use these credentials to authenticate the following nodes:
OpenWeatherMap
Supported authentication methods
#
API access token
Related resources
#
Refer to
OpenWeatherMap's API documentation
for more information about the service.
Using API access token
#
To configure this credential, you'll need an
OpenWeatherMap
account and:
An
Access Token
To get your
Access Token
:
After you verify your email address, OpenWeatherMap includes an
API Key
in your welcome email.
Copy that key and enter it in your n8n credential.
If you'd prefer to create a new key:
To create a new key, go to
Account >
API Keys
.
In the
Create Key
section, enter an
API Key Name
, like
n8n integration
.
Select
Generate
to generate your key.
Copy the generated key and enter it in your n8n credential.
