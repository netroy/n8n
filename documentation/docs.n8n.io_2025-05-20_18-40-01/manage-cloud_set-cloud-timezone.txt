Set the Cloud instance timezone
#
You can change the timezone for your n8n instance. This affects the
Schedule Trigger
and
Date & Time node
. Users can configure the timezone for individual workflows in
Workflow settings
.
On your dashboard, select
Manage
.
Change the
Timezone
dropdown to the timezone you want.
