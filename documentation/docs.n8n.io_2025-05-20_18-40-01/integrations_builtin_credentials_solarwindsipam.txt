SolarWinds IPAM credentials
#
You can use these credentials to authenticate when using the
HTTP Request node
to make a
Custom API call
.
Supported authentication methods
#
Username & Password
Related resources
#
Refer to
SolarWinds IPAM's API documentation
for more information about the service.
Using Username & Password
#
To configure this credential, you'll need a SolarWinds IPAM account and:
URL
: The base URL of your SolarWinds IPAM server
Username
: The username you use to access SolarWinds IPAM
Password
: The password you use to access SolarWinds IPAM
Refer to
SolarWinds IPAM's API documentation
for more information about authenticating to the service.
