Security Assertion Markup Language (SAML)
#
Feature availability
Available on Enterprise plans.
You need access to the n8n instance owner account to enable and configure SAML
Available from version 0.225.0.
This section provides guidance on using SAML SSO (single sign-on) with n8n. It assumes you're familiar with SAML. If you're not,
