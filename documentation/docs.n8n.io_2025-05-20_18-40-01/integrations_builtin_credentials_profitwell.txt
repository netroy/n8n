ProfitWell credentials
#
You can use these credentials to authenticate the following nodes:
ProfitWell
Prerequisites
#
Create a
ProfitWell
account.
Supported authentication methods
#
API token
Related resources
#
Refer to
Profitwell's API documentation
for more information about the service.
Using API token
#
To configure this credential, you'll need:
An
API Token
: To get an API key or token, go to
Account Settings > Integrations
and select
ProfitWell API
.
