Datadog credentials
#
You can use these credentials to authenticate when using the
HTTP Request node
to make a
Custom API call
.
Prerequisites
#
Create a
Datadog
account.
Related resources
#
Refer to
Datadog's API documentation
for more information about authenticating with the service.
This is a credential-only node. Refer to
Custom API operations
to learn more. View
example workflows and related content
on n8n's website.
Using API Key
#
To configure this credential, you'll need:
Your Datadog instance
Host
An
API Key
An
App Key
Refer to
Authentication
on Datadog's website for more information.
