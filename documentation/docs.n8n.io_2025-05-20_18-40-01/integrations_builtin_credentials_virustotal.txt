VirusTotal credentials
#
You can use these credentials to authenticate when using the
HTTP Request node
to make a
Custom API call
.
Prerequisites
#
Create a
VirusTotal
account.
Supported authentication methods
#
API key
Related resources
#
Refer to
VirusTotal's documentation
for more information about the service.
This is a credential-only node. Refer to
Custom API operations
to learn more. View
example workflows and related content
on n8n's website.
Using API key
#
To configure this credential, you'll need:
An
API Token
: Go to your
user account menu > API key
to get your API key. Enter this as the
API Token
in your n8n credential. Refer to
API authentication
for more information.
