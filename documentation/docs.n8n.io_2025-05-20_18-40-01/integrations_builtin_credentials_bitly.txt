Bitly credentials
#
You can use these credentials to authenticate the following node:
Bitly
Prerequisites
#
Create a
Bitly
account.
Supported authentication methods
#
API token
OAuth2
Related resources
#
Refer to
Bitly's API documentation
for more information about the service.
Using API token
#
To configure this credential, you'll need:
An
Access Token
: Once logged in, visit
Settings > Developer Settings > API
to generate an Access Token.
Using OAuth2
#
Note for n8n Cloud users
Cloud users don't need to provide connection details. Select
Connect my account
to connect through your browser.
If you need to configure OAuth2 from scratch or need more detail on what's happening in the OAuth web flow, refer to the
Bitly API Authentication documentation
for more information.
