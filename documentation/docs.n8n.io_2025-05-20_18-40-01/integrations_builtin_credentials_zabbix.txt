Zabbix credentials
#
You can use these credentials to authenticate when using the
HTTP Request node
to make a
Custom API call
.
Prerequisites
#
Create a
Zabbix Cloud
account or self-host your own Zabbix server.
Supported authentication methods
#
API key
Related resources
#
Refer to
Zabbix's API documentation
for more information about the service.
This is a credential-only node. Refer to
Custom API operations
to learn more. View
example workflows and related content
on n8n's website.
Using API key
#
To configure this credential, you'll need:
an
API Token
: An API key for your Zabbix user.
the
URL
: The URL of your Zabbix server. Don't include
/zabbix
as part of the URL.
Refer to
Zabbix's API documentation
for more information about authenticating to the service.
