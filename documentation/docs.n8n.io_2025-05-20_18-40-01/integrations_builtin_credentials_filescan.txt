Filescan credentials
#
You can use these credentials to authenticate when using the
HTTP Request node
to make a
Custom API call
.
Prerequisites
#
Create a
Filescan
account.
Related resources
#
Refer to
Filescan's API documentation
for more information about authenticating with the service.
This is a credential-only node. Refer to
Custom API operations
to learn more. View
example workflows and related content
on n8n's website.
Using API key
#
To configure this credential, you'll need:
An
API Key
: Generate your API key from your
profile settings
> API Key
. Refer to the
Filescan FAQ
for more information.
