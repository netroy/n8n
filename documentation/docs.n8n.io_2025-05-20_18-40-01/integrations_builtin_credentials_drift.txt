Drift credentials
#
You can use these credentials to authenticate the following nodes:
Drift
Prerequisites
#
Create a
Drift
account.
Create a Drift app
.
Supported authentication methods
#
API personal access token
OAuth2
Related resources
#
Refer to
Drift's API documentation
for more information about the service.
Using API personal access token
#
To configure this credential, you'll need:
A
Personal Access Token
: To get a token,
create a Drift app
.
Install the app
to generate an OAuth Access token. Add this to the n8n credential as your
Personal Access Token
.
Using OAuth2
#
Note for n8n Cloud users
Cloud users don't need to provide connection details. Select
Connect my account
to connect through your browser.
If you need to configure OAuth2 from scratch or need more detail on what's happening in the OAuth web flow, refer to the instructions in the
Drift Authentication and Scopes documentation
to set up OAuth for your app.
