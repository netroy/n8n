Figma credentials
#
You can use these credentials to authenticate the following nodes:
Figma Trigger (Beta)
Prerequisites
#
Create a
Figma
account. You need an admin or owner level account.
Supported authentication methods
#
API key
Related resources
#
Refer to
Figma's API documentation
for more information about the service.
Using API key
#
To configure this credential, you'll need:
A Personal
Access Token
(PAT): Refer to the
Figma API Access Tokens documentation
for instructions on generating a Personal
Access Token
.
