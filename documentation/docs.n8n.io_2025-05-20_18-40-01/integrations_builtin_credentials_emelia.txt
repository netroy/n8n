Emelia credentials
#
You can use these credentials to authenticate the following nodes:
Emelia
Emelia Trigger
Prerequisites
#
Create an
Emelia
account.
Supported authentication methods
#
API key
Related resources
#
Refer to
Emelia's API documentation
for more information about the service.
Using API key
#
To configure this credential, you'll need:
An
API Key
: To generate an API Key in Emelia, access your
API Keys
by selecting the avatar in the top right (your
Settings
). Refer to the Authentication section of
Emelia's API documentation
for more information.
