Cisco Umbrella credentials
#
You can use these credentials to authenticate when using the
HTTP Request node
to make a
Custom API call
.
Prerequisites
#
Create a
Cisco DevNet developer account
.
A
Cisco Umbrella user account
with
Full Admin
role.
Authentication methods
#
API key
Related resources
#
Refer to
Cisco Umbrella's API documentation
for more information about the service.
This is a credential-only node. Refer to
Custom API operations
to learn more. View
example workflows and related content
on n8n's website.
Using API key
#
To configure this credential, you'll need:
An
API Key
A
Secret
: Provided when you generate an API key
Refer to the
Cisco Umbrella Manage API Keys documentation
for instructions on creating an Umbrella API key.
