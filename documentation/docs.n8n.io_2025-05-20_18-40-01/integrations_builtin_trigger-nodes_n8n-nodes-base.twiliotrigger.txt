Twilio Trigger node
#
Use the Twilio Trigger node to respond to events in
Twilio
and integrate Twilio with other applications. n8n has built-in support for a wide range of Twilio events, including new SMS and calls.
On this page, you'll find a list of events the Twilio Trigger node can respond to and links to more resources.
Credentials
You can find authentication information for this node
here
.
Examples and templates
For usage examples and templates to help you get started, refer to n8n's
Twilio integrations
page.
Events
#
On New SMS
On New Call
New Call Delay
It can take Twilio up to thirty minutes to generate a summary for a completed call.
