ConvertKit credentials
#
You can use these credentials to authenticate the following nodes:
ConvertKit
ConvertKit Trigger
Prerequisites
#
Create a
ConvertKit
account.
Supported authentication methods
#
API key
Related resources
#
Refer to
ConvertKit's API documentation
for more information about the service.
Using API key
#
To configure this credential, you'll need:
An
API Secret
: Access your ConvertKit API key in
Account Settings > Advanced
. Add this key as the
API Secret
in n8n.
