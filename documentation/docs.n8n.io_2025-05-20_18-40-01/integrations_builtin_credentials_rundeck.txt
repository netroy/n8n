Rundeck credentials
#
You can use these credentials to authenticate the following nodes:
Rundeck
Prerequisites
#
Create a user account on a
Rundeck
server.
Supported authentication methods
#
API token
Related resources
#
Refer to
Rundeck's API documentation
for more information about the service.
Using API token
#
To configure this credential, you'll need:
Your
URL
: Enter the base URL of your Rundeck server, for example
http://myserver:4440
. Refer to
URLs
for more information.
A user API
Token
: To generate a user API token, go to your
Profile > User API Tokens
. Refer to
User API tokens
for more information.
