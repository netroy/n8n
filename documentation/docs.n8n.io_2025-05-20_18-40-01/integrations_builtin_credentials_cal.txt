Cal.com credentials
#
You can use these credentials to authenticate the following nodes:
Cal.com Trigger
Prerequisites
#
Create a
Cal.com
account.
Supported authentication methods
#
API key
Related resources
#
Refer to
Cal.com's API documentation
for more information about the service.
Using API key
#
To configure this credential, you'll need:
An
API Key
: Refer to the
Cal API Quick Start documentation
for information on how to generate a new API key.
A
Host
: If you're using the cloud version of Cal.com, leave the Host as
https://api.cal.com
. If you're self-hosting Cal.com, enter the
Host
for your Cal.com instance.
