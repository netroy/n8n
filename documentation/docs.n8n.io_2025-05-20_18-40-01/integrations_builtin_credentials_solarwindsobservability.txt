SolarWinds Observability SaaS credentials
#
You can use these credentials to authenticate when using the
HTTP Request node
to make a
Custom API call
.
Supported authentication methods
#
API Token
Related resources
#
Refer to
SolarWinds Observability SaaS's API documentation
for more information about the service.
Using API Token
#
To configure this credential, you'll need a SolarWinds Observability SaaS account and:
URL
: The URL you use to access the SolarWinds Observability SaaS platform
API Token
: An API token found in the SolarWinds Observability SaaS platform under
Settings > Api Tokens
Refer to
SolarWinds Observability SaaS's API documentation
for more information about authenticating to the service.
