Freshservice credentials
#
You can use these credentials to authenticate the following nodes:
Freshservice
Prerequisites
#
Create a
Freshservice
account.
Supported authentication methods
#
API key
Related resources
#
Refer to
Freshservice's API documentation
for more information about the service.
Using API key
#
To configure this credential, you'll need:
An
API Key
: Refer to the
Freshservice API authenticaton documentation
for detailed instructions on getting your API key.
Your Freshservice
Domain
: Use the subdomain of your Freshservice account. This is part of the URL, for example
https://<subdomain>.freshservice.com
. So if you access Freshservice through
https://n8n.freshservice.com
, enter
n8n
as your
Domain
.
