Built-in methods and variables
#
n8n provides built-in methods and variables for working with data and accessing n8n data. This section provides a reference of available methods and variables for use in
expressions
, with a short description.
Availability in the expressions editor and the Code node
Some methods and variables aren't available in the Code node. These aren't in the documentation.
All data transformation functions are only available in the expressions editor.
The
Cookbook
contains examples for some common tasks, including some
Code node only
functions.
