#

Segment

#

Segment
account.

#
API key
Related resources
#

#
To configure this credential, you'll need:
A
Write Key
: To get a Write Key, go to
Sources > Add Source
. Add a
Node.js
source and copy that write key to add to your n8n credential.
Refer to
Locate your Write Key
for more information.
