Unleashed Software credentials
#
You can use these credentials to authenticate the following nodes:
Unleashed Software
Prerequisites
#
Create an
Unleashed Software
account.
Supported authentication methods
#
API key
Related resources
#
Refer to
Unleashed's API documentation
for more information about the service.
Using API key
#
To configure this credential, you'll need:
An
API ID
: Go to
Integrations > Unleashed API Access
to find your
API ID
.
An
API Key
: Go to
Integrations > Unleashed API Access
to find your
API Key
.
Refer to
Unleashed API Access
for more information.
Account owner required
You must log in as an Unleashed account owner to view the API ID and API Key.
