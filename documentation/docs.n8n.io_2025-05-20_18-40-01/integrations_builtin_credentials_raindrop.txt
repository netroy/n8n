Raindrop credentials
#
You can use these credentials to authenticate the following nodes:
Raindrop
Prerequisites
#
Create a
Raindrop
account.
Supported authentication methods
#
OAuth2
Related resources
#
Refer to
Raindrop's API documentation
for more information about the service.
Using OAuth
#
To configure this credential, you'll need:
A
Client ID
A
Client Secret
Generate both by creating a Raindrop app.
To create an app, go to
Settings >
Integrations
and select
+ Create new app
in the
For Developers
section.
Use these settings for your app:
Copy the
OAuth Redirect URL
from n8n and add it as a
Redirect URI
in your app.
Copy the
Client ID
and
Client Secret
from the Raindrop app and enter them in your n8n credential.
