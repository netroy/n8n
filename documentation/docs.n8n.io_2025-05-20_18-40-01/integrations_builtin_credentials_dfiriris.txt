DFIR-IRIS credentials
#
You can use these credentials to authenticate when using the
HTTP Request node
to make a
Custom API call
.
Prerequisites
#
An accessible instance of
DFIR-IRIS
.
Related resources
#
Refer to
DFIR-IRIS's API documentation
for more information about authenticating with the service.
This is a credential-only node. Refer to
Custom API operations
to learn more. View
example workflows and related content
on n8n's website.
Using API Key
#
To configure this credential, you'll need:
An
API Key
: Refer to
DFIR-IRIS's API documentation
for instructions on getting your API key.
The
Base URL
of your DFIR-IRIS instance.
